# Understanding-customers
# Customer Segmentation with K-means & RFM Analysis 

## Project Objective
Segment e-commerce customers using **RFM analysis** and **K-means clustering** to help define targeted marketing strategies.

## Methods
- **Data cleaning and preprocessing**: remove missing values, handle cancellations, correct prices.  
- **RFM feature engineering**: calculate Recency, Frequency, and Monetary metrics.  
- **Outlier management**: manual segmentation for high-value customers.  
- **Clustering**: K-means to group similar customers.  
- **Visualization**: 3D scatter plots and violin plots to analyze clusters.

## Tools & Libraries
- **Python**: Pandas, NumPy  
- **Machine Learning**: Scikit-learn (K-means)  
- **Visualization**: Matplotlib, Seaborn  
- **Other**: Jupyter Notebook / Google Colab
